# Catalog Service Search
Cache Service of [catalog service](https://github.com/ahlp/catalog-service) in Haskell

# How To Build

## Image 
 - make build
 - make run

## Service
 - cd node node-catalog-search-proxy/ 
 - node server.js
 - cd ../
 - runghc Main.hs
